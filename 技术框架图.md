## 🔄 三、核心功能数据流图

### 3.1 智能搜题功能流程

```
用户上传图片
    ↓
┌─────────────────────────────────────────────────────────────┐
│  app.py: render_smart_upload()                               │
│  • st.file_uploader() 接收图片                               │
│  • 显示上传进度                                              │
└───────────────────────┬─────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────────┐
│  ai_utils.py: ocr_extract_text()                            │
│  • 图片转Base64编码                                          │
│  • 调用 SiliconFlow API (Qwen2-VL-72B)                     │
│  • 返回LaTeX格式的题目文本                                   │
└───────────────────────┬─────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────────┐
│  app.py: 显示OCR识别结果                                      │
│  • st.write() 显示识别文本                                   │
└───────────────────────┬─────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────────┐
│  ai_utils.py: solve_problem_text_stream()                    │
│  • 调用 SiliconFlow API (Qwen2.5-32B)                       │
│  • 流式返回解题步骤（实时显示）                               │
│  • 解析JSON格式（analysis, answer, subject, topic, kp）    │
└───────────────────────┬─────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────────┐
│  app.py: 实时显示解题过程                                     │
│  • st.write_stream() 流式显示                                │
│  • clean_latex() 清洗LaTeX格式                               │
│  • 显示学科、主题、知识点                                     │
└───────────────────────┬─────────────────────────────────────┘
                        ↓
用户点击"保存到错题库"
    ↓
┌─────────────────────────────────────────────────────────────┐
│  db_manager.py: add_question()                              │
│  • 连接SQLite数据库                                          │
│  • 插入questions表                                          │
│  • 返回question_id                                          │
└───────────────────────┬─────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────────┐
│  app.py: 显示保存成功                                         │
│  • st.success() 提示信息                                     │
│  • 更新界面                                                  │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 沉浸式复习功能流程

```
用户进入复习模式
    ↓
┌─────────────────────────────────────────────────────────────┐
│  db_manager.py: get_questions_due_for_review()             │
│  • 查询next_review_time <= 当前时间的题目                    │
│  • 按记忆曲线算法筛选                                        │
│  • 返回需要复习的题目列表                                     │
└───────────────────────┬─────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────────┐
│  app.py: render_review_mode()                               │
│  • 显示题目（文本+图片）                                      │
│  • 提供答题输入框（文本/图片）                                │
└───────────────────────┬─────────────────────────────────────┘
                        ↓
用户提交答案
    ↓
┌─────────────────────────────────────────────────────────────┐
│  ai_utils.py: judge_answer_stream()                         │
│  • 如果用户提交图片，先OCR识别                                │
│  • 提取标准答案（extract_answer_from_solution）             │
│  • 调用 SiliconFlow API (Qwen2.5-32B) 判卷                 │
│  • 流式返回判卷结果（is_correct, reason）                   │
└───────────────────────┬─────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────────┐
│  app.py: 实时显示判卷结果                                     │
│  • st.write_stream() 流式显示评语                            │
│  • 显示正确答案和解析                                         │
└───────────────────────┬─────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────────┐
│  db_manager.py: update_question_mastery()                   │
│  • 更新correct_count或wrong_count                           │
│  • 计算next_review_time（记忆曲线算法）                      │
│  • 插入review_logs表（记录复习日志）                         │
└───────────────────────┬─────────────────────────────────────┘
                        ↓
生成同类题（可选）
    ↓
┌─────────────────────────────────────────────────────────────┐
│  ai_utils.py: generate_similar_question_stream()            │
│  • 调用 SiliconFlow API (Qwen2.5-32B)                       │
│  • 流式生成相似题目                                           │
└─────────────────────────────────────────────────────────────┘
```

### 6.1 记忆曲线算法（艾宾浩斯遗忘曲线优化版）

```
用户答对/答错题目
    ↓
┌─────────────────────────────────────────────────────────────┐
│  判断答题结果                                                 │
└───────────┬─────────────────────────────────────────────────┘
            │
    ┌───────┴───────┐
    │               │
答错              答对
    │               │
    ↓               ↓
┌─────────┐   ┌─────────────────────────────────────────────┐
│ 10分钟后│   │ 根据correct_count计算下次复习时间              │
│ 重新复习 │   │                                              │
└─────────┘   │ correct_count = 0 → 1天                     │
              │ correct_count = 1 → 7天                     │
              │ correct_count = 2 → 15天（达到掌握）         │
              │ correct_count >= 3 → 15天（保持）            │
              └─────────────────────────────────────────────┘
```

```

    ↓

│          yield chunk.choices[0].delta.content               │
└───────────────────────┬─────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────────┐
│  app.py: st.write_stream()                                  │
│  • 实时显示每个chunk                                         │
│  • 用户看到逐字显示效果                                       │
└─────────────────────────────────────────────────────────────┘
```

## 📝 总结

### 技术架构特点

1. **一体化框架**：Streamlit将前端和后端合并，简化开发
2. **模块化设计**：通过函数和模块分离职责，易于维护
3. **流式优化**：AI输出采用流式显示，提升用户体验
4. **轻量级部署**：SQLite数据库，无需额外服务
5. **外部AI服务**：通过API调用，无需本地部署模型

### 技术栈总结

- **前端**：Streamlit + Plotly + CSS
- **后端**：Python + SQLite
- **AI能力**：SiliconFlow API (Qwen2-VL-72B + Qwen2.5-32B)
- **数据存储**：SQLite数据库
- **部署方式**：本地运行（streamlit run app.py）
