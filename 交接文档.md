```
│   ├── render_dashboard()     # 首页仪表盘

│   └── render_learning_stats()# 学习统计

├── requirements.txt
├── mistakes.db         # SQLite 数据库（已在 .gitignore）
├── .env                # API Key（已在 .gitignore）
└── CLAUDE.md           # 项目协作规则
```

## 6. 关键技术决策

- **AI 调用**：SiliconFlow API（Qwen 系列），超时友好的中文错误提示
- **状态管理**：Streamlit session_state，双阶段状态机（answering / feedback）
- **复习算法**：SM-2 变体，答错 10 分钟 → 答对1次 1天 → 答对2次 7天 → 答对3次 15天归档
- **数据隔离**：通过 user_id 实现多用户数据隔离
